
Tedesco Facile - Mini corso (Lezioni 1-10)
----------------------------------------

Questo pacchetto è un sito statico (HTML/JS) che usa la Web Speech API per TTS e localStorage per flashcard/SRS.
Puoi aprire index.html direttamente nel browser per provarlo (su alcuni browser TTS funziona meglio servendolo via HTTP, non file://).

Per ospitarlo online (scelta rapida):
1) GitHub Pages:
   - Crea un nuovo repository su GitHub e carica i file in questo pacchetto.
   - Vai su Settings -> Pages e scegli il branch 'main' (o 'gh-pages') e cartella '/'.
   - Dopo pochi minuti avrai un link tipo https://tuo-nome.github.io/nome-repo/

2) Netlify (drag & drop):
   - Vai su https://app.netlify.com/drop
   - Trascina la cartella estratta (contiene index.html, lessons.json, package.zip).
   - Netlify pubblicherà il sito e ti darà un URL.

3) Vercel o altri host statici funzionano allo stesso modo.

Note su TTS:
- La voce TTS è fornita dal browser (Web Speech API). Chrome desktop e Edge offrono voci 'de-DE' buone.
- Su iOS Safari e alcuni browser mobili il supporto può essere limitato.

Buon apprendimento! Se vuoi, posso preparare io il repository GitHub pronto e fare il deploy per te (avrei bisogno di un repo o di autorizzazione per creare uno repo sotto il tuo account), oppure posso fornirti istruzioni passo-passo scritte.
